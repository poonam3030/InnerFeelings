Inner Feelings – Mental Health Care Web Application: 
Inner Feelings is a mental health care web application designed to provide emotional support resources online.  
The platform allows users to access mental wellness content, share personal recovery experiences and book appointments - all from the comfort of their home. This project focuses on accessibility, awareness, and creating a supportive digital environment using web technologies.

Project Overview: The goal of this project is to build a centralized web platform where users can:
1. Access mental health educational content
2. Share recovery stories in a safe space
3. Book appointments for support services
4. Explore calming resources and freebies
The application emphasizes simplicity, accessibility, and user comfort.

Technical Architecture
Frontend Layer : Responsible for user interaction and interface
1. HTML – Structure and page layout
2. CSS – Styling, responsiveness, UI design  
3. JavaScript – Dynamic content, form handling and user interaction  

Key responsibilities:
1. Login/registration interface
2. Blog and story display
3. Appointment booking interface
4. Resource/freebie display

Application Logic Layer: Handles the core functionality
1. User authentication logic
2. Content rendering
3. Form validation
4. Interactive features

Backend language — PHP, Python

Data Layer : Stores application data such as
1. User account details
2. Blog posts and recovery stories
3. Appointment booking data
4. Resource/freebie information

Database used — MySQL

Key Features
1. Secure user login and registration
2. Mental health blog section
3. Recovery story sharing
4. Appointment booking functionality
5. Free calming resources for users
6. Simple and user-friendly interface

Learning Outcomes : Through this project I gained experience in
1. Designing a user-focused web application
2. Structuring multi-feature web platforms
3. Handling authentication flows
4. Improving UI clarity and accessibility

Project Vision: Inner Feelings aims to reduce barriers to mental health support by offering accessible digital resources in a safe, supportive environment.
The long-term goal is to expand this platform into a more comprehensive mental wellness ecosystem.
